# COMM3800
## Walk for Green Plus

<context>
I'm a non-technical founder building a smart campus mobility phone app for CUHK students. It modernizes the university's "Walk for Green" program by seamlessly syncing with native health apps (Apple Health, Google Fit, WeChat Sports), using GPS background check-in to verify on-campus walking, and featuring an elevation map and weekly leaderboard. I have a tight deadline (launching by April 10) and a strict $0 budget. I need beginner-friendly research with actionable insights.
</context>

<instructions>

1. What similar gamified fitness or campus apps exist, and what mechanics do they use to keep users engaged?

3. What completely free no-code/low-code tools are best for this specific use case (including frontend, database, and backend)?
4. How can I easily integrate a campus elevation map without coding from scratch?
5. What AI tools can accelerate development so I can successfully launch by April 10?

### Research Focus:
- Simple, actionable insights with examples tailored strictly to a $0 budget.

- Examples of similar successful university or gamified fitness projects.

### Required Deliverables:

3. **MVP Implementation Plan** — How to realistically build the 3 core features without extensive coding.
4. **Development Roadmap** — A rapid 3-week timeline to hit the April 10 deadline.

</instructions>
